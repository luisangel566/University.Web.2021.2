# Sistema de Gestión Académica - ASP.NET MVC

## Descripción

Aplicación web desarrollada en ASP.NET MVC para la gestión de estudiantes y procesos académicos, implementando arquitectura en capas para separar la lógica de negocio y la presentación.

## Arquitectura

* University.Web: Capa de presentación (MVC)
* University.BL: Lógica de negocio

## Tecnologías

* C# (.NET Framework)
* ASP.NET MVC
* SQL Server
* HTML, CSS, JavaScript

## Funcionalidades

* Gestión de estudiantes
* Operaciones CRUD
* Lógica de negocio estructurada

## Cómo ejecutar

1. Abrir el archivo University.sln en Visual Studio
2. Configurar la conexión a la base de datos
3. Ejecutar el proyecto

## Objetivo

Aplicar buenas prácticas de desarrollo utilizando arquitectura en capas y el patrón MVC.
